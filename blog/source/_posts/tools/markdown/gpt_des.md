基于OpenAI提供的API开发的**第三方**ChatGPT的客户端。 
{% note simple blue %}
这意味这你无需注册OpenAI的账号以及身处国外就能够使用ChatGPT。 
{% endnote %}
当然，如果你拥有OpenAI的账号，你就可以拥有**独立**的Openai API Key。使用他们公司提供的API。
**ChatGPT的使用**是完全免费的。
<br>
{% note simple blue %}
但是**ChatGPT API**的使用是付费的。所以这些网站也会有一笔不小的运营成本。 
{% endnote %}
